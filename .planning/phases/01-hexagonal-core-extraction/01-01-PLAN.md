# Plan 01-01: Extract price parsing, CSV inventory, and history loading into shared modules

<objective>
Extract the current CSV discovery, retailer detection, price normalization, source-specific row normalization, and bounded price-history loading behavior from `streamlit_app.py` into framework-independent core modules. The extracted code must preserve the current normalized history DataFrame contract and `(history_df, skipped_df)` loader return shape while removing Streamlit dependencies from the loading path. This plan covers decisions D-01, D-02, D-09, D-10, D-13, D-14, D-15, and D-16 from `01-CONTEXT.md`.
</objective>
